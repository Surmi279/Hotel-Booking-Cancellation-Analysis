# Hotel Booking Cancellation Analysis

## Project Overview
This project analyzes hotel booking cancellation trends using Microsoft Excel. The dataset was cleaned, processed, and analyzed to identify factors affecting booking cancellations.

## Objectives
- Analyze hotel booking cancellation trends
- Understand customer booking behavior
- Compare cancellation rates across guest types and room categories
- Create an interactive dashboard for visualization

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- IF + AND formulas
- Conditional Formatting
- Slicers
- Data Cleaning
- Dashboard Creation

## Analysis Performed
- Guest Type vs Cancellation
- Customer Type vs Cancellation
- Room Type vs Cancellation
- Booking Channel Analysis
- Cancellation Percentage Analysis

## Insights
- Identified guest categories with higher cancellation rates
- Compared room types contributing to more cancellations
- Analyzed booking channels with maximum cancellations
- Visualized trends using Pivot Tables and charts
